TeX macro packages

The central location for TeX macro packages is the Comprehensive TeX Archive Network (CTAN, http://www.ctan.org/). You should look there or on journal websites for appropriate macro packages. arXiv does not provide for distribution of an up to date collection of macro packages.

Historical collection of 1990's macro packages

The following is an unmaintained random collection of mostly user contributed macros from the 1990's which is kept for historical reasons. Most of these macros are out of date and you should instead download current macros from CTAN. Please do not ask us to update this list.
